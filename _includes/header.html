Course Communication

We will be using Slack for sending announcements, collecting homeoworks, providing students with feedback, and answering question. Slack is a cloud-based team collaboration platform that supports many app integrations such as Dropbox, Google Drive, Hangouts, Trello, and Skype.

Software

We will mainly be using R and RStudio for homework assignments. Since the university provides software licenses for SPSS Amos, students can use Amos as an aid in completing the course assignments and also for fitting models discussed in class. Homeworks will be provided as R Markdown documents that you will need to weave into html documents.

Grading

Grades are calculated in accordance with student-oriented principles. Each student is encouraged to compete with himself/herself and not with each other. Class average is not computed. Homework scores are transformed into an integer from 3 to 5. For each assignment, a threshold for thorough understanding is computed, and each student who scores above that threshold earns a 5. Students who score above the threshold for adequate understanding earn a 4. Lower than that threshold gets you a 3. This grading scheme deemphasizes grades and reduces the need for grade grubbing. It also puts more weight on understanding the material. Grades will be posted on Carmen Canvas.
